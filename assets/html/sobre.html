Sobre o Projeto

Esse é um projeto de "To do list". Consiste em criar uma aplicação Web que salve no localStorage do navegador o texto criado pelo usuário através de um input junto com um ID para cada texto. Cada ID gerado será aleatório e não se repetirá a fim de evitar Bugs no futuro. Cada tarefa nova criado pelo usuário gerará uma elemento HTML novo através do Javascript.

Linguagens utilizadas:

  • HTML
  • CSS
  • Javascript

Notas:

Esse projeto possuí fins totalmente didáticos.
